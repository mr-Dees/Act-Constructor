# Папка 2 — Корректор, режим «Улучшить читаемость»

Здесь **нет исходников D17**, и это ожидаемо.

Промпт этого режима (`READABILITY_SYSTEM_PROMPT`) написан на нашей стороне как
рабочая основа (baseline): по просьбе владельца D17 передала готовым только промпт
орфографии (папка `1`), а промпт читаемости попросила написать самим и допилить
позже. Подробности — в инструкции `2 — Корректор (улучшение читаемости).md`.

Папка существует, чтобы нумерация наработок совпадала с нумерацией инструкций:

- `1/` — корректор «Исправить ошибки» (исходник D17: `orphography_v2.py`).
- `2/` — корректор «Улучшить читаемость» (исходников D17 нет — промпт наш baseline).
- `3/` — формализация нарушения (исходник D17: конвейер разбора описания).

Источник истины по промпту читаемости —
`app/domains/chat/services/text_actions/prompts.py` (`READABILITY_SYSTEM_PROMPT`).
